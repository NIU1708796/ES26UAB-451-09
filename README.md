# QueMenges - Proyecto de Ingeniería del Software (2025-2026)

## Descripción del proyecto
**QueMenges** es una aplicación innovadora diseñada para permitir a los usuarios acceder a menús preparados por personas individuales, fomentando la comida casera y de proximidad. A diferencia de otras plataformas de entrega de comida de grandes superficies, QueMenges permite que cualquier persona pueda ofrecer sus platos y recetas, mientras que los usuarios pueden elegir en tiempo real dentro de una gran variedad de opciones.

Un mismo usuario puede asumir de forma dinámica los roles de **Cliente, Cocinero o Repartidor** según sus necesidades.

## Estructura del repositorio
Este repositorio sigue la estructura obligatoria definida para el desarrollo del proyecto mediante la metodología Scrum:

* **`/documents/`**: Contiene los documentos finales generados en cada Sprint (como el PDF de Especificaciones - S.R.S.).
* **`/requirements/`**: Almacena las fichas individuales de cada uno de los requisitos del sistema en formato Markdown (`.md`).
* **`/minutes/`**: Contiene las actas de las diferentes reuniones del equipo y con el tutor, formateadas en Markdown.
* **`/others/`**: Directorio reservado exclusivamente para plantillas, borradores o archivos adicionales que no pertenecen a las categorías anteriores.
