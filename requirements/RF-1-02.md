# Requirements Template

### ID: 
RF-1-02

### Type/Category:
Funcional / Perfil de Usuario

### Version: 
1.0

### Title: 
Perfil de Usuario y Restricciones Alimentarias

### Description: 
El sistema requerirá que los usuarios al registrarse rellenen un perfil indicando su nombre de usuario y su dirección de recogida antes de realizar un pedido. Opcionalmente, permitirá añadir restricciones alimentarias (alergias, alimentos no deseados) para que la aplicación filtre automáticamente los menús que pueden consumir.

### Relations: 
Sense completar (pending)

### Comments: 
El nombre de usuario debe tener 5 carácteres como mínimo y no puede estar repetido en la BD.
