# Requirements Template

### ID: 
RNF-6-02

### Type/Category:
NO Funcional / Trazabilidad y Auditoría

### Version: 
1.0

### Title: 
Registro de auditoría (logs) del sistema de fidelización

### Description: 
El sistema deberá generar y mantener un registro interno inalterable de todos los movimientos de recompensas o ventajas asociadas a las suscripciones mensuales de los usuarios, registrando fecha, hora, ID de usuario y la acción realizada.

### Relations: 
Sense completar (pending)

### Comments: 
Requisito técnico indispensable para que los administradores puedan rastrear y resolver posibles incidencias o quejas de los usuarios premium
